Historia del HTML5:

Cronología Completa de HTML

Esta sección presenta una línea de tiempo detallada que documenta la evolución de HTML desde su concepción inicial como un sistema simple de intercambio de documentos científicos hasta su transformación en el lenguaje fundamental para aplicaciones web modernas. Cada versión representa un salto significativo en capacidades y alcance.

Evolución Cronológica de HTML

HTML 1.0 (1991-1993):

Año: 1991.
Creador: Tim Berners-Lee en el CERN.

Características Clave:

  • Conjunto muy limitado de 18 etiquetas.
  • Etiquetas básicas: <html>, <head>, <title>, <body>
  • Soporte para encabezados (<h1> a <h6>)
  • Párrafos (<p>) y saltos de línea (<br>).
  • Hipervínculos (<a>) - la innovación principal.
  • Listas ordenadas y desordenadas.
  • Sin soporte para imágenes o tablas.

HTML 2.0 (1995):

Año: 1995.
Organismo: IETF (Internet Engineering Task Force).

Características Clave:

  • Primera especificación oficial estándar (RFC 1866).
  • Introducción de formularios (<form>, <input>, <textarea>, <select>).
  • Soporte para imágenes (<img>).
  • Tablas básicas (<table>, <tr>, <td>).
  • Etiquetas de formato de texto (<b>, <i>, <u>).
  • Atributos como align, bgcolor.

HTML 3.2 (1997):

Año: 1997.
Organismo: W3C (World Wide Web Consortium).

Características Clave:

  • Tablas complejas con colspan y rowspan.
  • Soporte para applets de Java (<applet>).
  • Scripts del lado del cliente (<script>).
  • Hojas de estilo CSS (limitado).
  • Frames básicos.
  • Mayor control sobre el diseño visual.

HTML 4.01 (1999):

Año: 1999.
Organismo: W3C.

Características Clave:

  • Tres variantes: Strict, Transitional, Frameset.
  • Separación completa de contenido y presentación.
  • Mejor soporte para CSS.
  • Mejoras en accesibilidad.
  • Frames mejorados (<iframe>).
  • Objetos embebidos (<object>).

XHTML 1.0 (2000):

Año: 2000.
Organismo: W3C.

Características Clave:

  • Reformulación de HTML 4.01 como aplicación XML.
  • Sintaxis más estricta (etiquetas en minúsculas).
  • Todas las etiquetas deben cerrarse.
  • Atributos deben estar entrecomillados.
  • Elementos vacíos deben auto-cerrarse (<br />).
  • Mejor integración con XML.

HTML5 (2014):

Año: 2014 (recomendación oficial).
Organismo: W3C y WHATWG.

Características Clave:

  • Elementos semánticos (<header>, <nav>, <article>, <section>, <footer>).
  • Multimedia nativa (<audio>, <video>, <canvas>).
  • Gráficos vectoriales escalables (<svg>).
  • APIs JavaScript avanzadas (Geolocalización, Web Storage, Web Workers).
  • Formularios mejorados (nuevos tipos de input).
  • WebSockets para comunicación bidireccional.
  • Eliminación de elementos obsoletos (<font>, <center>, <frame>).
